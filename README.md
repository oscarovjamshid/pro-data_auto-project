# PRO-DATA AUTOMATION APP
Using tools like Cypress and Mochawesome Reporter

# RUNNING APP
- npm run test          Start running scripts via terminal
- npm run dev           Start running scripts via Cypress UI

# INSTRUCTIONS TO INSTALL APP

```bash
# Клонировать репозиторий
git clone https://github.com/username/project-name.git

# Перейти в директорию проекта
cd pro-data

# Установить зависимости
npm install
```
