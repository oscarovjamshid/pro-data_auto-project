# Подготовка к запуску существующего приложения Cypress

## Установка зависимостей

1. Убедитесь, что у вас установлены Node.js и npm. Вы можете скачать их с [официального сайта Node.js](https://nodejs.org/).

2. Откройте терминал и перейдите в корневую директорию вашего проекта.

3. Установите все необходимые зависимости, выполнив команду:
    ```
    npm install
    ```

## Запуск тестов

Все необходимые данные для запуска авто тестов находятся в файле /cypress/fixtures/examples.json
Подготовка окружении для запуска авто тестов:
1. Должно бьть подготовлен 1 новый аккаунт для запуска авто тестов (данные аккаунта должно прописатся на файле /cypress/fixtures/examples.json)
2. Должно быть подготовлен 1 другой аккаунт для проверки блокировка аккаунта (данные аккаунта должно прописатся на файле /cypress/fixtures/examples.json)
3.Должно быть подготовлен 1 другой аккаунт для проверки истекшего аккаунта - Expired User test flow (данные аккаунта должно прописатся на файле /cypress/fixtures/examples.json)
4. Должно быть сервер с называнием Test-Server (Если будете менять, нужно изменить на файле /cypress/fixtures/examples.json)
5. После созданые сервера ID созданного сервера (нужно изменить на файле /cypress/fixtures/examples.json)
6. В странице Диски нужно создать новый диск с называнием Test-Disk (Если будете менять, нужно изменить на файле /cypress/fixtures/examples.json)
7. В странице Сети нужно создать 2 сети с называнием  NTester и Test_Local_Ip (Если будете менять, нужно изменить на файле /cypress/fixtures/examples.json)
8. В странице Сети нужно создать public IP с называнием  NTesterUchun 
9. Перед запуском необходимо очистить все сети, диски, бэкапы, расписании бэкапов и поставить статус сервера на "Остановлен"

1. Для запуска Cypress в режиме интерфейса пользователя, выполните команду:
    ```
    npx cypress open
    ```
    или
    ```
    npm run dev
    ```

2. Для запуска Cypress в режиме командной строки, выполните команду:
    ```
    npx cypress run
    ```
    или
    ```
    npm run test
    ```
